# PartielReact

Création d'un jeu du pendu.

## Pour le fonctionnement

### Etape 1 :

Faire un git clone https://github.com/Eclipse94w/PartielReact

### Etape 2 :

Faire un `npm install`

### Etape 3 :

Faire un `npm start`

## Et voilà, maintenant vous pouvez jouer au jeu du pendu.
